<p>本作は、人気ゲーム『原神』の世界観とキャラクター設定を基にした、ファンメイドの二次創作テキストアドベンチャーゲームです。</p>

- オリジナル脚本と事件設計。心を揺さぶる推理とサスペンスのストーリー展開
- 調査、推理、そして法廷審理を通じ、事件に覆いかぶさる謎の霧を次々と払拭していく
- 7～8時間にわたる没入型の完全体験
- カスタムメイドの美麗なキャラクター立ち絵や差分、約30枚のストーリーCG
- 全キャラクター・全メインストーリーにフルボイス対応
- メインストーリーの進行に応じて解放されるサブミニゲーム「海亀スープ」

<br/>
<img src="docs/介绍小标题_剧情介绍_jp.png" width="420px">
<p>旅人であるあなたとナヒーダはスメールの危機を解決し、その後、ナヒーダは国の統治に忙しい日々を送るようになります。彼女の努力により、スメールの国は日々良くなっていきます。しかし、新たな危機が再び訪れました。<br/>
頻発する地震が、ティワット全土を飲み込む未曽有の災害を予告しています。ナヒーダは「世界樹」が再び病に侵されているのではないかと推測しますが、手立てがありません。<br/>
そんな中、奇妙な感覚に導かれた彼女は、とある神秘的な部屋へと足を踏み入れます……</p>

<p>ある日、あなたとパイモンはナヒーダからの休暇の招待状を受け取ります。アルハイゼンやディシアと共に奇妙な「缶詰知識」に触れ、その中に保存されていた「過去の」スメールの都へと足を踏み入れることになります。<br/>
しかし、花神誕日の祭典で次々と怪異が巻き起こります。不可思議な夢、奇妙な殺人事件、見覚えがあるようで見知らぬ顔、「神」が存在しない国……。「ここは本当にスメールなのか？」<br/>
未知の世界を探求し、殺人事件の犯人と地震の謎を解明してください。そして、「あの彼女」と再び巡り会い……<br/>
最後に世界を救い、「世界樹」とティワット全体の秘密を発見してください。</p>
