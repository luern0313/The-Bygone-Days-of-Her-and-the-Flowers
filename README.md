<img src="docs/介绍小标题_关于本游戏_zh.png" width="420px">
<p>本作是一款基于人气游戏《原神》世界观与角色设定，二次创作的粉丝向文字冒险游戏。</p>

- 原创剧本与案件设计，动人心魄的推理与悬疑剧情
- 通过调查、推理与庭审，层层拨开案件的迷雾
- 7至8小时的全流程沉浸式剧情体验
- 定制的精美人物立绘与差分，近30张剧情CG
- 全人物、全主线流程配音
- 随主线推进解锁的支线「海龟汤」小游戏

<br/>
<img src="docs/介绍小标题_剧情介绍_zh.png" width="420px">
<p>旅行者纳西妲一众人解决须弥城的危机后，纳西妲一直忙于对国家的治理，在她的努力下，须弥每一天都在变好。<br/>
然而，新的危机却再次降临。频繁的地震，预示着一场史无前例的灾难将吞噬整个提瓦特。纳西妲猜测，「世界树」再次陷入了病痛。<br/>
束手无策之时，一股奇异的感应引导她走向了一个神秘的房间…</p>

<p>某日，你与派蒙收到了纳西妲的度假邀请函。你们与艾尔海森和迪希雅一起，触碰了一个奇怪的罐装知识，进入了度假的目的地——罐装知识存储着的「过去的」须弥城。<br/>
但花神诞祭庆典时，种种怪异事件也接踵而至。诡异的梦境、离奇的命案，熟悉但又陌生的面孔、「没有神明」的国度……「这里…真的是须弥吗？」<br/>
探索未知的世界，寻找命案的凶手、地震频发的真相；还有，与「那个人」再次相见…<br/>
最后，拯救世界，发掘世界树和整个提瓦提的秘密。</p>
